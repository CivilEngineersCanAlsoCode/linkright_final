# Linkright Quick Start Guide

Welcome to **Linkright (lr) v4.0**, an enterprise-grade agentic ecosystem powered by BMAD v6.0.

## 1. System Activation

To begin, interact with the **LR-Orchestrator** using the following protocol:

- **Command**: `[RS] Resume Session` to pick up your latest active Bead context.
- **Discovery**: Use `/lr-menu` at any time to return to the global module selector.

## 2. Configuration & Identity

Your professional identity is governed by `_lr/lr-config.yaml`.

- Ensure your `user_name`, `bio`, and `target_role` are accurately populated.
- Agents use these session variables to calibrate all generated artifacts.

## 3. Core Workflows

- **Outreach**: Run `[GO] Sync` -> `outbound-campaign` to generate signal-driven career artifacts.
- **Amplification**: Run `[GO] Flex` to transform career signals into viral social content.
- **Enterprise Delivery**: Run `[GO] Squick` for high-velocity requirement decomposition.

## 4. Governance & Integrity

Every action is tracked in the **Beads** ledger.

- Use `bd ready` to see your next available tasks.
- Always run `bd sync` before concluding a session to secure memory persistence.
