# Step 01: Session Setup

## Objective

Establish the context, goals, and methodologies for the brainstorming session.

## Instructions

1.  **Define Goal**: Clearly state the problem or opportunity being addressed.
2.  **Select Techniques**: Choose from available techniques (e.g., Mind Mapping, Reverse Thinking, SCAMPER).
3.  **Identify Participants**: Specify if any specific Linkright agents should be consulted (optional integration with Party Mode).
4.  **Verification**: Ensure the setup is documented in the session context.

## Deliverables

- `brainstorming_goal`: Structured objective for the session.
