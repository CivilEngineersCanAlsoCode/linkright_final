# Step 03: Graceful Exit

**Goal:** Conclude the session, summarize findings, and clear the discussion context.

---

## 1. Final Summary

- **The Orchestrator** must provide a consolidated briefing of the session's results.
- List all major decisions, "Intel" gathered, or "Missions" defined.

## 2. Agent Sign-off

- Each participant provides a one-sentence sign-off in their specific persona tone.

## 3. Context Flush

- Clear any temporary party-mode states.
- Revert agents to their default operating modes.

---

## MISSION COMPLETE

The party mode has concluded. Returns control to the main system.
