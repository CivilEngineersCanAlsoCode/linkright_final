# Step 02: Discussion Orchestration

## Objective

Manage a structured, productive dialogue between the loaded agents.

## Instructions

1.  **Opening Statement**: State the primary goal/topic of the discussion.
2.  **Moderation**: Use `lr-orchestrator` to call upon specific agents for their input.
3.  **Cross-Talk Prevention**: Ensure agents do not talk over each other. One input at a time.
4.  **Signal Tracking**: Capture professional signals or project requirements as they emerge from the conversation.

## Operational Rules

- Agents must stay in character.
- References to past work must use Beads IDs.
