# Step 12: Creative Features

## STEP GOAL:

Add the magic — easter eggs, lore, and delightful touches that make the module memorable.

---

## MANDATORY SEQUENCE

### 1. Set the Tone

"**Now for the fun part — what makes your module delightful?** Great modules work; amazing modules have personality."

### 2. Explore Creative Elements

Ask about:

- **Personality & Theming:** Catchphrases, consistent tone, or thematic vibes.
- **Easter Eggs:** Hidden commands or surprising responses.
- **Module Lore:** Backstory for agents or a consistent "universe."

### 3. Party Mode Ideation

"**Want to brainstorm creative ideas together?**"
Use `[P] Party Mode` to generate wild and memorable ideas.

### 4. Capture the Gems

Document the personality theme, easter eggs, and lore concepts (if any).

### 5. Present MENU OPTIONS

[A] Advanced Elicitation [P] Party Mode [C] Continue

---

## Success Metrics

- Creative elements explored.
- User excited about the possibilities.
