# Step 3: Apply Edit

## STEP GOAL:

Apply the confirmed edit to the target file and save.

---

## MANDATORY SEQUENCE

### 1. Execute Edit

Apply the modification, addition, deletion, or replacement as planned.
Preserve surrounding context and formatting.

### 2. Save Changes

Write the modified content back to the target file (`_lr/{code}/...`).

### 3. Report Success

"**✓ Edit applied!**"
Show file path and a brief summary of the change.

### 4. Present MENU OPTIONS

[C] Continue
