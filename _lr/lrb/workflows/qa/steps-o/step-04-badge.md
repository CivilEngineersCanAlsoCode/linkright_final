# Step 04: Generate Badge

**Workflow**: Quality Gate Orchestrator
**Agent**: QA Engineer

---

## 🎯 Goal

Generate the final Quality Badge and summary for the module's documentation.

---

## 📋 Instructions

1.  **Load Score**: READ the final score and status from Step 03.
2.  **Generate Markdown Badge**: Create a status badge using Shields.io format (e.g., `![Status](https://img.shields.io/badge/Quality-PASS-green)`).
3.  **Finalize Validation Report**: APPEND the detailed results and score to the target's `validation-report.md`.
4.  **Update README**: Inject the Quality Badge into the top of the module's `README.md`.
5.  **Register Status**: Update `module.yaml` with the latest `quality_score` and `last_tested_at` timestamp.
