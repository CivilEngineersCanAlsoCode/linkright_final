# Step 02: Generative Ideation

**Goal:** Generate 50+ divergent ideas across different professional domains.

---

## 1. Divergence Round

- Each agent must provide 5-10 ideas in their specific domain.
- **Rules**: No critique, build on each other's ideas, go for high quantity.

## 2. Domain Pivoting

- After 20 ideas, **The Orchestrator** must force a pivot (e.g., "Now, how would a UX designer view this?").
- Use orthogonal categories to break semantic drift.
