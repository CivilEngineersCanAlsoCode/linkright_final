# Step 01: Agent Loading

**Goal:** Formally introduce the selected participants and set the context for the session.

---


## DEPENDENCIES
- Requires: Previous step output (if sequential)
- Requires: `lr-config.yaml` session context

## 1. Introduction

- Present a "Loading Intel" briefing.
- List each participating agent with their Name, Role, and Icon (from `agent-manifest.csv`).

## 2. Shared Context Briefing

- Summarize the user's request.
- Define a "Session Objective" that all agents will work toward.

## 3. Personality Activation

- State that all agents are now in "Merged Persona Mode" for this discussion.
- Each agent should acknowledge the objective in their specific tone.
