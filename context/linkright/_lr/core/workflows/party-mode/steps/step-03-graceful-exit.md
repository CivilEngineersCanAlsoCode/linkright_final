# Step 03: Graceful Exit


## DEPENDENCIES
- Requires: Previous step output (if sequential)
- Requires: `lr-config.yaml` session context

## Objective

Finalize the session and ensure all outcomes are documented.

## Objective

1.  **Summary**: Provide a bulleted summary of the key discussion points and decisions.
2.  **State Update**: Update or create Beads tasks for any action items identified.
3.  **Cleanup**: Unload transient session variables.
4.  **Closing**: Formally end the party and return to the main orchestrator menu.

## Deliverables

- `session_summary.md`: Documented outcome of the party.
