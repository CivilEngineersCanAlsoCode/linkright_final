# Content Automation Quality Checklist

## Pre-Publication (10 checks)

- [ ] Content aligns with user's professional brand
- [ ] No confidential information disclosed
- [ ] Grammar and spelling verified (automated + manual)
- [ ] Links are valid and tested
- [ ] Images/media optimized for platform
- [ ] CTA is clear and actionable
- [ ] Tone matches audience expectations
- [ ] Post is original (plagiarism check)
- [ ] Platform formatting correct (hashtags, threads, etc.)
- [ ] Scheduled for optimal timing (≥9am local, weekday bias)

## Post-Publication (7 checks)

- [ ] Post published successfully without errors
- [ ] Initial engagement metrics captured (first 2 hours)
- [ ] Typos or errors caught and corrected
- [ ] Comments moderated and responded to
- [ ] Metrics logged to dashboard
- [ ] Cross-promotion complete (other platforms)
- [ ] Learnings captured for next post

## Weekly Review (5 checks)

- [ ] Weekly engagement report generated
- [ ] Top performing topics identified
- [ ] Content gaps identified
- [ ] Audience feedback synthesized
- [ ] Next week's content planned

## Success Metrics
- Engagement rate: >2% (target 3-5%)
- Share rate: >0.5%
- Comment sentiment: >70% positive
- Follower growth: +2-5% weekly
- Content consistency: >80% on-time publication
