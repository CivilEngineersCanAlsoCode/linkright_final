# Step 9: Workflows

## STEP GOAL:

Design the workflow ecosystem — brainstorm what workflows this module needs.

---

## MANDATORY SEQUENCE

### 1. Brainstorm Workflows

"**What workflows should your module have?**"
Categorize into:

- **Core Workflows:** Essential functionality.
- **Feature Workflows:** Specialized capabilities.
- **Utility Workflows:** Supporting operations.

### 2. For Each Workflow

Capture:

- **Name & Purpose.**
- **Input → Process → Output.**
- **Owning Agent.**

### 3. Workflow Connections

"**How do workflows connect?**"
Sequence and dependencies between workflows.

### 4. Present MENU OPTIONS

[A] Advanced Elicitation [P] Party Mode [C] Continue

---

## Success Metrics

- Workflow list generated.
- Agent-workflow mappings defined.
- Flow understood.
