# Step 11: Scenarios

## STEP GOAL:

Tell the story of how the module works in practice.

---

## MANDATORY SEQUENCE

### 1. Set the Stage

"**Let's bring your module to life with some scenarios.** How does it handle real-world tasks?"

### 2. Guide the User Through Scenarios

Ask:

- "**Scenario 1: The 'Quick Win'** — What's a simple, common task this module handles perfectly?"
- "**Scenario 2: The 'Complex Challenge'** — How does the agent team collaborate on something difficult?"
- "**Scenario 3: The 'Surprising Capability'** — What's something users wouldn't expect it to do?"

### 3. Capture the Stories

Summarize the 2-3 usage scenarios. "**Do these stories accurately reflect your module's potential?**"

### 4. Present MENU OPTIONS

[A] Advanced Elicitation [P] Party Mode [C] Continue

---

## Success Metrics

- 2-3 usage scenarios documented.
- User can visualize the module in action.
