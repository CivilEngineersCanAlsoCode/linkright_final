# Phase L: Styling & Theming - Design System Application

## MANDATORY EXECUTION RULES (READ FIRST)
✅ ALWAYS apply company theming consistently.
🛑 NEVER override design guidelines arbitrarily.

## DEPENDENCIES
- Requires: `step-54-layout-check` output (content_layout_validated.yaml)
- Requires: Company branding guidelines

## EXECUTION PROTOCOLS
1. [READ] Load content and company branding/styling guide.
2. [THEME] Apply visual styling:
   - Color scheme (primary, accent, highlight colors)
   - Typography (fonts, weights, sizes for headings/body)
   - Spacing (margins, padding, line-height)
   - Visual hierarchy (sizing, emphasis, contrast)
3. [COMPILE] Generate styled HTML/PDF portfolio output.
4. [VALIDATE] Review styling for:
   - Brand alignment
   - Readability
   - Visual impact
   - Format compliance (PDF dimensions, etc.)
5. [OUTPUT] Generate styled_portfolio.html with design applied.

## OUTPUT
- styled_portfolio.html: Themed and styled portfolio
- Next step: step-56-final-quality

## NOTES
- Professional design dramatically impacts first impression
- Consistency across document signals attention to detail
- Company branding guidelines must be followed
