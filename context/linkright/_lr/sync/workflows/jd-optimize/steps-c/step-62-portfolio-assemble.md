## MANDATORY EXECUTION RULES (READ FIRST)
✅ ALWAYS validate final portfolio before delivery.
🛑 NEVER send portfolio with formatting issues.

## DEPENDENCIES
- Requires: `step-61-content-refine` output (content_refined.yaml)
- Requires: `step-55-styling` output (styled_portfolio.html)

## EXECUTION PROTOCOLS
1. [READ] Load refined content and styled portfolio template.
2. [ASSEMBLE] Generate final portfolio output:
   - Integrate all content into portfolio format
   - Apply final styling and branding
   - Verify all sections present and complete
   - Check formatting consistency throughout
3. [VALIDATE] Quality check:
   - No spelling or grammar errors
   - All formatting consistent (fonts, spacing)
   - Links functional (if applicable)
   - PDF/HTML renders correctly
4. [EXPORT] Create multiple formats (PDF, HTML, plain-text).
5. [OUTPUT] Generate final_portfolio files in multiple formats.

## OUTPUT
- final_portfolio.pdf: Delivery-ready PDF portfolio
- final_portfolio.html: Web-viewable portfolio
- final_portfolio.txt: Plain-text version
