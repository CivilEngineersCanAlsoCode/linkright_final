# Outbound Campaign Checklist

Ensuring every outreach touchpoint is surgical, authentic, and character-compliant.

## 1. Research Phase

- [ ] Recruiter/Hiring Manager PDF or Profile loaded.
- [ ] Psychological "Hook" identified (The Bridge).
- [ ] Tone selection (Formal/Casual/Technical) approved by user.

## 2. Drafting Phase

- [ ] Cover Letter drafted with Synergy Triggers.
- [ ] In-Mail drafted specifically for the Hiring Manager.
- [ ] Connection Invite validated to be ≤ 300 characters.
- [ ] All drafts follow the user's authentic "Voice".

## 3. Brand Phase

- [ ] Social profile update suggestions generated.
- [ ] Campaign output file updated with `stepsCompleted` metadata.
