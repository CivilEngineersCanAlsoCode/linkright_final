# Quick-Optimize Workflow Instructions

This is the Linkright Quick-Flow accelerator for minor resume iterations. It avoids the full 53-step heavy data structuring pipeline in favor of a fast 5-step rapid iteration process with built-in adversarial review.

## Architecture

- **Phase 1**: Ingest diff instruction
- **Phase 2**: Rapid edit
- **Phase 3**: Adversarial Review
- **Phase 4**: Self-check against checklist
- **Phase 5**: Finalize output

Follow the sequence in `steps-c/` strictly.
