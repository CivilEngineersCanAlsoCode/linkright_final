# Signal Capture Workflow

Capture, structure, and validate raw professional career signals.

## Sequence:
1. **Capture**: Ingest raw professional data (resume, LinkedIn, projects).
2. **Extract**: Map to the career signal ontology (tech stack, accomplishments, identity).
3. **Validate**: Ensure XYZ format compliance for every metric.

## Output:
- `career_signals.yaml` — Structured signal package.
