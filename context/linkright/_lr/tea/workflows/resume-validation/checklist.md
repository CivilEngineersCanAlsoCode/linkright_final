# Resume Quality Audit Checklist

## 1. Professional Ontology Compliance

- [ ] Are P0 skills clearly mapped to accomplishments?
- [ ] Is the XYZ (Action, Metric, Scaling) format strictly followed?
- [ ] No generic "soft skills" list without evidence.

## 2. ATS Readiness

- [ ] High-density keyword placement in first 33% of page?
- [ ] Standard header hierarchy (H1 for name, H2 for sections)?
- [ ] Clean dates and location formatting?

## 3. Narrative Resonance

- [ ] Does the profile summary have a clear "Hook"?
- [ ] Is the technical growth trajectory evident across roles?
