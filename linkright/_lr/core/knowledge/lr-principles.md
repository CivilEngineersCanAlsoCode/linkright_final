# Linkright Principles

1. **Zero Monoliths**: Logic must be isolated into module-specific steps.
2. **Signal Gravity**: Metadata (signals) must inform all generation (Sync/Flex).
3. **Hub-and-Spoke Connectivity**: Cross-module communication only via the Core registry.
4. **Branding Integrity**: Zero BMAD identity. Always "Linkright (lr)".
5. **Session Continuity**: Every workflow must support resumption via Beads.
