# Task: Help

Generate or display a comprehensive help mapping for Linkright users.

## Global Commands

- `/lr-menu`: Return to the main orchestrator menu.
- `/lr-help`: Show global documentation and command overview.
- `/lr-status`: Show the project health status (Beads summary).

## Module Helper Commands

- `/sync-help`: Detailed guide for the Sync signal processing module.
- `/flex-help`: Detailed guide for the Flex content automation module.
- `/squick-help`: Detailed guide for the Squick developer-first module.

## Instructions

- Always display commands in a clean, scannable table or list.
- If a specific module is active, prioritize that module's commands.
