# Step 6: Users

## STEP GOAL:

Define who the module is for AND how they'll use it — personas and user journey combined.

---

## MANDATORY SEQUENCE

### 1. Define the Users

"**Let's get specific about who this is for.**"

- **Primary Users:** Who will use this most? (role/skill level)
- **Secondary Users:** Who else benefits?

### 2. Build User Personas

Create 1-2 brief personas:

- **Persona 1:** Name/role, goals, pain points.
- **Persona 2:** Name/role, goals, pain points.

### 3. Tell the User Journey Story

"**Let's walk through how someone would use this module.**"

1. User has a problem.
2. They load the module.
3. They run an agent/workflow.
4. They get a result.
5. This helps them achieve X.

"**Does this match what you envision?**"

### 4. Present MENU OPTIONS

[A] Advanced Elicitation [P] Party Mode [C] Continue

---

## Success Metrics

- User personas defined.
- User journey story told.
