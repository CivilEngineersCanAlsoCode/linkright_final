# Step 1: Load Target (Validate Mode)

## STEP GOAL:

Load the target module or component for validation and initialize the report.

---

## MANDATORY SEQUENCE

### 1. Determine Validation Target

"**What would you like to validate?**"
Options:

- **[M]odule** — Complete module structure.
- **[B]rief** — Module brief.
- **[A]gents** — Agent specifications/files.
- **[W]orkflows** — Workflow specifications/files.

### 2. Load Target

Locate the target directory or file (`_lr/{code}/` or creations folder).

### 3. Initialize Validation Report

Create the structured report to track all PASS/FAIL/WARNING results.

[C] Continue
