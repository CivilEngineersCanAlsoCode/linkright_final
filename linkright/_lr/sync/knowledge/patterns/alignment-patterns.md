# JD Alignment Patterns

## 1. ATS Matching Logic

- Keywords must appear in context, not just lists.
- Semantic similarity is weighted higher than exact string matching for seniority roles.

## 2. Signal Weighting

- **Type A (Explicit)**: Direct match (e.g., "5 years React"). Weight: 40%.
- **Type B (Implicit)**: Narrative match (e.g., "Led team of 10"). Weight: 40%.
- **Type C (Company Culture)**: Cultural synergy. Weight: 20%.

## 3. The "Gold Standard" Rez

- Metric Density: >2 metrics per bullet.
- Star Clarity: Situation -> Task -> Action -> Result (with Linkright Synergy).
