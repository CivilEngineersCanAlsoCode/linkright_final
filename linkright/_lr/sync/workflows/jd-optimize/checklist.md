# JD Optimization Checklist

Use this checklist to ensure every optimized JD profile meets the Linkright quality standards.

## 1. Ingestion Phase

- [ ] Raw JD text captured completely.
- [ ] Core Role Identity (Job Title) identified.
- [ ] Target Company context loaded.

## 2. Analysis Phase

- [ ] P0 Tech Requirements identified.
- [ ] P0 Power Requirements identified.
- [ ] All "Golden Signals" from user history matched.
- [ ] Gaps between user history and JD requirements noted.

## 3. Output Phase

- [ ] Optimized profile preserves user's authentic voice.
- [ ] Signal alignment is clear and impact-driven.
- [ ] Template `optimized-jd.template.md` fully populated.
- [ ] Final document includes `stepsCompleted` metadata.
